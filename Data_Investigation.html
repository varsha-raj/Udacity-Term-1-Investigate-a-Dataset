Analysis for Study Question 1¶

Genres is a very important characteristic in movies. Genres help classify movies into a specific category. In most cases however, movies are generally a blend of multiple genres. For example a romantic movie can also be comedy. Such movies usually comes under Romantic/Comedy genre.

For this experiment, to keep the analysis simple and straightforward, five broad sectors of genre were used to classify movies and are listed below.

GENRE LIST: 1:Fiction 2:Thriller 3:Romance 4:Comedy 5:Horror

Movies that are a blend of mutpliple genres are shrunk to a single genre category by matching based on the rank order of genre list. This means that if a movie is classified as Thriller/Fiction/Drama, then this movie falls under the 'Fiction' category. If a movie is classified as Horror/Comedy, then this movie falls under 'Comedy' category.

Only a subset of movie data that contain one or more of these five broad genre types are used in this analysis. Movies that do not include any of the genres listed above are ignored.

The genre list is stored as a python list. A list is an ordered collection of items and each item/element in the list has an index value (0,1,2,3,...,n). The index vales specifies the order of the items in the list.

The test aimed to create a linear regression model for revenue (dependent variable) as a function of popularity and vote counts (independent variables).

Multiple Linear Regression is used when two or more predictors (independent variables) are used to asses the the outcome variable (dependent variable).
In order to develop a mulpiple linear regression model, scatter plots were created to first check for linear correlation between the dependent variable (revenue) and the independent variables (popularity and vote count).

For simple or multiple regression analysis, linear relationshiop between the dependent and independent variables is important. From the two scatter plots, revenue seems to be linearly associated with both vote counts and popularity. For a multiple linear regression model, another important criteria is that the predictor variables should be independent of each other. In case of strong correlation between the predictors, the predictor that has the least influence on the outcome variables is dropped from the analysis. Variation Inflation factors (VIF) was used to test the presence of mulicollinearity.

Limitations:¶

The bar plots indicating distribution of genres across months uses movie data from not just Hollywood but also other international movies. The analysis does not specify the breakdown between the Hollywood movies and international movies. Also the bar plots indicate the trends across months for only five broad genre types.

The line plots and regression model is representative of only one production company. It is also important to note that in certain cases multiple production companies jointly participate in producing a movie. This analysis for simplicity purposes extracts any data that comes under Warner Bros banner.

When building regression models, general practise is to not use the entire dataset, instead the complete dataset is broken down into K subsets. Regression model is fit on the training dataset (k-1) and the predictions from the model is applied to the test (last subset) dataset to assess the accuracy of the model. This method is referred to as the K-Fold Cross Validation technique. In this analysis however, since a single production company data was used as a case study to assess the relationship between two variables, building the model for machine learning didn't seem necessary.
